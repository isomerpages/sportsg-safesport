<h3><strong>The Safe Sport Unified Code</strong></h3>
<p>The Unified Code was launched on 23 November 2021. It is the cornerstone
of the Safe Sport Programme and sets out a unified reference for the Singapore
sporting community that defines, describes and explains the forms of abuse
and harassment which may take place in the sporting environment.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/codebook.png">

<p>The Unified Code code is <strong>free-to-use</strong> and may be adopted
by any organisation.</p>
<p>See examples of how it is referenced in <a href="https://file.go.gov.sg/activesg-terms-of-use.pdf" rel="noopener noreferrer nofollow" target="_blank">facility use terms</a>,
event <a href="https://www.bacfurious5s.com/tournament/house-rules" rel="noopener noreferrer nofollow" target="_blank">codes of conduct</a> and
organisational <a href="https://www.safesport.sg/policy-and-research/resources/" rel="noopener noreferrer nofollow" target="_blank">policies</a>.</p>
<p>Categories of misconduct under the Unified Code:</p>
<ul data-tight="true" class="tight">
<li>
<p>Offences Under Domestic Laws</p>
</li>
<li>
<p>Sexual Misconduct</p>
</li>
<li>
<p>Physical and Psychological Misconduct</p>
</li>
<li>
<p>Other Inappropriate Conduct</p>
</li>
<li>
<p>Misconduct related to Process</p>
</li>
</ul>
